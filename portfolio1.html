School Insight

In this project I have built a pipeline of opensource data technologies to exploit the wealth of publically available information on schools.

I use this project as a platform to practice my data developer skillset. Data wrangling with Python, NoSQL databases, web frameworks, searchable data indices, JavaScript visualisation, even a bit of Hadoop (in the form of Pig).

My current subproject is to include data pulled from the Ofsted website by subjecting it to some simple Natural Language processing.

  • The pipeline I have built draws largely upon one I picked up from Agile Big Data by Russell Jurney.

    Data is downloaded or pulled from the web using Python and wrangled into a more acceptable format. Hadoop, in the form of Pig, uploads it to a MongoDB database generating some interesting analytics as it goes. The result is displayed on a web page using a lightweight web framework.
