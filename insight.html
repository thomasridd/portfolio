School Insight

Data product development

Edu-data pipeline

School Insight is my platform for experimenting with new technology

The principle is very simple. Begin with as much data as possible. Subject it to a battery of OpenSource technology. Terminate with a browser based data product. Iterate.

Architecture

Initial architecture owes a lot to Russell Jurney's book Agile Big Data

  • Python + Java: Acquire and wrangle data
  • Hadoop (Pig): Batch process data
  • MongoDB: Store data
  • Python + Flask: Application server
  • HTML + Bootstrap: User interface
  • ElasticSearch: Search functionality
  • D3.js: Visualizations
