WikiWhere

iOS Data Driven App

iOS development

I like programming for iOS. You can knock up quality prototypes very quickly but further development requires creative thinking and intellectual application which is satisfying on multiple levels.

My first app was a quite tame mapping tool for iPad. With my second I decided to ring the changes with a more sophisticated iPhone app that exploited a technology I found particularly interesting at the time, the semantic web.

Architecture

The app prompts for a location then, using a selection of sources (one fast, one thorough), picks up nearby geocoded Wikipedia articles

  • Platform: Objective-C for iOS
  • Geolocation: Apple MapKit & Googlemaps API
  • Datasource 1: Semantic Web: DbPedia
  • Datasource 2: Semantic Web: YAGO
  • Datasource 3: REST api GeoSearch
  • Monetization: iAd + in-app purchase
