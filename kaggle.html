Cats and Dogs

Practical machine learning

Kaggle.com

Kaggle.com's cats vs dogs contest required creation of an algorithm that could classify pet images as cats or dogs.

Taking my first steps in machine learning with unstructured image data was ambitious. Over-ambitious as it turns out. Still I'm a big believer in learning through doing and I learned more in this contest than any that I actually completed.

Approach

My main problem was sheer volumne of images. I needed to process them heavily so I came up with a plan...

  • Process the images
  • Pick out keypoints
  • Neural network to get only animal pieces.
  • Another neural network to classify cat or dog

    • Java + OpenCV + R + R. Easy

    Results

    83% accuracy to classify a piece as animal or not animal.

    Unfortunately a neural network was just too slow and with deadline looming I had to face the fact that my little computer wouldn't be up to the job even if I were
