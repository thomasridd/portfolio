KAGGLE Cats and Dogs

Skills Used: Java - R - OpenCV - Machine Learning - PCA - Neural Networks

Kaggle.com sets up competitions for the machine learning community. The aim of the cats and dogs competition was to create a classifier algorithm that could distinguish a photo of a cat from one of a dog.

The Java, OpenCV and R neural network hybrid I trained could filter a picture down to areas that contained an animal and those that didn't had an 86% accuracy rate.
