LaSwap Self Entry

Web app development

Self Entry Data Collection

LaSwap admissions week is intense and data collection is old school paper forms.

I floated an idea to collect last minute information with a web app which I then built with some very happy results.

Architecture

During a hackathon I picked up a working knowledge of Google's AppEngine and decided to put this to use.

  • Data storage: Google DataStore
  • Cloud PaaS: Google App Engine
  • Application server: Python
  • Front End: HTML + css + JavaScript

    • HTML & Javascript experience derives from the School Insight project also in this portfolio

    Design

    A pleasing aspect of this project was working with my colleagues on the basic concept and on how the app would sit as a component of the wider admissions day system

    I also worked closely with a UX designer which improved the flow of the app considerably

    More...

    LaSwap 2014 worked first time without hitch and is still in use as a reference tool. Really it has been an unqualified success that I am very proud of.

  • Python: Exams day without a safety net
  • R: Progress and batch calculation
