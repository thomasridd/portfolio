  • The challenge of WikiWhere was detail versus speed when it came to pulling data.

    WikiLocation is a REST API that is fast, reliable, and easy to use. DBPedia and the Yago project are two semantic web Wikipedia projects. They are slow, unreliable, complete pigs to use but provide more detailed results.

    In the end I run a race with results from all three.
